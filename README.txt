Copy either HMCFAMain.py & HMCFAPhi4UC.py for fourier accelerated HMC or HMCCSFMain.py & HMCCSFUC.py for standard HMC along with unewfuncs into a folder.
Run in terminal as applies to your installation of python and choice of module: python HMCFAMain.py

Data is printed in terminal for various observables.
Some plots are generated, check code for format/location.
